THE GRID OF COVID


This project aims to study “Why there is a need for order in the Grid System?”

It is a comparative and experimental research examined by using ‘Grid System’ in Newspapers, as a metaphor, to this COVID-19 lockdown scenario and animating the grids according to different global issues recognised. The theory of two core fields of design i.e. modernism design and postmodernism design are applied to indicate the difference evolved up. These animations are thus presented in an interactive website where the user gets to select the issue and experience the world news being out of order, indirectly concluding the importance and need for order. This is an allegory example of Design Research which is to try to look at current designs, which in case is Grid System, and see how they work within a natural environment.
